Welcome to playdoh’s documentation!¶

Mozilla’s Playdoh is a web application template based on Django.

Patches are welcome! Feel free to fork and contribute to this project on Github.

Features¶

Quick and dirty (and probably incomplete) feature list:

  • Django
  • SHA-512 passwords
  • X-Frame-Options: DENY by default
  • Celery support
